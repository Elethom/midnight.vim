# midnight.vim

Xcode Midnight color scheme for VIM.

## Install

### Terminal

Open `Midnight.terminal` and set it as default profile.

### VIM

If you use vim-plug: `Plug 'Elethom/midnight.vim'`

And don't forget to add `colorscheme midnight`.
